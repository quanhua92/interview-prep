# Frontend Engineer -- Prep Checklist

Use this checklist to track your preparation progress across core frontend engineering competencies. Aim to complete every item before your interview window opens.

## Topic Prep Checklist per Competency

- [ ] **JavaScript & TypeScript Fundamentals**: Review closures, prototypal inheritance, event loop, promises, async/await, type systems (generics, union/intersection types, utility types), and TypeScript best practices.
- [ ] **React or Vue Proficiency**: Be prepared to explain component lifecycle, hooks (useState, useEffect, useMemo), state management patterns, and reconciliation.
- [ ] **CSS and Layout**: Practice flexbox, CSS Grid, the differences between Grid and Flexbox and when to use each, responsive design with media queries, and CSS-in-JS vs utility-first approaches (Tailwind).
- [ ] **Advanced UI Patterns**: Understand infinite scroll implementation (Intersection Observer, pagination strategies), virtualization for large lists, and optimistic UI updates.
- [ ] **Browser and DOM**: Understand the critical rendering path, DOM manipulation, reflow vs repaint, and the event delegation pattern.
- [ ] **Performance Optimization**: Study lazy loading, code splitting, tree shaking, web vitals (LCP, FID, CLS), and image optimization strategies.
- [ ] **Web Accessibility (a11y)**: Review ARIA roles, semantic HTML, keyboard navigation, screen reader basics, and WCAG guidelines.
- [ ] **State Management**: Be ready to discuss Context API, Redux, Zustand, or Pinia; understand when to use local vs global state.
- [ ] **Testing**: Practice writing unit tests (Jest, Vitest) and component tests (React Testing Library, Cypress) for UI components.

## Practice Questions

1. Explain the JavaScript event loop and how the call stack, microtasks, and macrotasks interact.
2. How does React reconciliation work and why are keys important?
3. What is the difference between useMemo, useCallback, and React.memo? When would you use each?
4. Describe the critical rendering path from receiving HTML to pixels on screen.
5. How would you optimize a React application that has become slow after adding many features?
6. Explain how CSS specificity works and how you would debug a specificity conflict.
7. What are closures and how are they used in real-world frontend applications?
8. How would you implement a debounced search input component?
9. Describe the difference between server-side rendering, client-side rendering, and static site generation.
10. How do you handle authentication and token management securely on the frontend?
11. What is the virtual DOM and how does it improve rendering performance?
12. How would you build an accessible modal dialog from scratch?
13. Explain cross-origin resource sharing (CORS) and why browsers enforce it.
14. What strategies would you use to reduce the bundle size of a large web application?
15. What are the differences between CSS Grid and Flexbox? When would you use each?
16. How would you implement infinite scroll in a news feed or product listing?
