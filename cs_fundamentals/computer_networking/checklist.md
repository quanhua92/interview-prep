# Computer Networking — Self-Assessment Checklist

Use this checklist to gauge your readiness for networking-related interview questions. Check off items as you can confidently explain them without notes.

## Concept Self-Assessment

- [ ] Can you explain the OSI model and TCP/IP model, and what each layer is responsible for?
- [ ] Can you describe the three-way handshake and four-way teardown in TCP?
- [ ] Can you explain the difference between TCP and UDP and when you would choose one over the other?
- [ ] Can you describe how DNS resolution works, including recursive and iterative queries?
- [ ] Can you explain what happens when a packet travels from one host to another on the same network vs. across networks?
- [ ] Can you explain HTTP/1.1, HTTP/2, and HTTP/3, including key differences and why HTTP/3 uses QUIC over UDP?
- [ ] Can you describe how TCP ensures reliable delivery, including sequence numbers, ACKs, sliding window, and congestion control?

## Explain-Out-Loud Prompts

- [ ] Explain to a non-technical friend what happens when you type google.com into the browser and press Enter.
- [ ] Describe how a load balancer works at layers 4 and 7, with use cases for each.
- [ ] Explain how TCP ensures reliable delivery using sequence numbers, acknowledgments, sliding window flow control, and congestion control algorithms.
- [ ] Walk through how TLS establishes a secure connection, including the role of certificates.

## Common Questions Practice

- [ ] What is the difference between a socket and a port?
- [ ] How does NAT work and why is it used?
- [ ] Explain the difference between ARP, ICMP, and DHCP protocols.
- [ ] What is a CDN and how does it improve website performance?
- [ ] How would you troubleshoot a slow network connection between two services?
