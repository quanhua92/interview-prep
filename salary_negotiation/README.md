# Salary Negotiation

Preparation for compensation negotiation at every stage of the interview process. Each topic has:

- **discussion.md** — Benchmarks, frameworks, scripts, and talking points
- **checklist.md** — Preparation steps, script practice, and scenario walkthroughs

## Topics

| Topic | Focus |
|-------|-------|
| [Market Research](market_research/) | Compensation data, leveling, and market positioning |
| [Total Comp Breakdown](total_comp_breakdown/) | Base, bonus, equity, benefits, and hidden value |
| [Initial Offer Evaluation](initial_offer_eval/) | Reading an offer letter, identifying negotiation levers |
| [Negotiation Tactics](negotiation_tactics/) | Anchoring, framing, timing, and communication strategies |
| [Equity & RSU Evaluation](equity_rsu_evaluation/) | Stock options, RSUs, vesting, strike price, and dilution |
| [Difficult Scenarios](difficult_scenarios/) | Lowball offers, exploding offers, competing offers, pay cuts |
| [Counteroffer & Staying](counteroffer_staying/) | When to accept a counteroffer, risks of staying |
| [Cross-Border Remote Negotiation](remote_negotiation/) | Remote pay models, employment structure, currency, benefits gaps for Vietnam-based engineers |

## How to Practice

1. Read the **discussion.md** once to learn the vocabulary and benchmarks
2. Open the **checklist.md** and work through the preparation steps
3. Practice scripts out loud — negotiation is a verbal skill
4. Role-play difficult scenarios with a partner
5. Use the practice log to track improvement
