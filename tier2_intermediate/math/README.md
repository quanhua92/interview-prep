# Math / Constructive

## Core Idea

Solve problems using mathematical formulas, number theory, or constructive approaches (building the answer piece by piece). These problems rely on mathematical insight rather than algorithmic patterns.

## When to Recognize

- "Palindrome product", "perfect square"
- "Poor pigs", "information theory"
- "Next greater element" by digit manipulation
- "Closest palindrome" — case analysis + construction
- "Random point in circle" — geometric probability

## Complexity

| Variant | Time | Space |
|---------|------|-------|
| Formula-based | O(1) or O(log n) | O(1) |
| Constructive enumeration | O(10^n) worst case | O(1) |
| Next permutation | O(n) | O(1) |
